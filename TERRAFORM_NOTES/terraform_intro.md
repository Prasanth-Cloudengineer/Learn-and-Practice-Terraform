

INTRODUCTION to TERRAFORM:

WHAT IS TERRAFORM?
Terraform is an open-source Infrastructure as Code (IaC) tool by HashiCorp that uses HashiCorp Configuration Language (HCL) to create and manage cloud and on-premises resources such as virtual machines, networks, databases.

WHAT IS IaC?
Terraform is an infrastructure as Code (IaC) tool used to create and manage on-premesis and cloud infrastructure using 
HashiCorp Configuration Language (HCL) in declarative way.

WHY TERRAFORM?
1. Cloud Agnostic
2. Multi Cloud support
3. State Management
4. Declarative
5. Shows Plan before Apply
